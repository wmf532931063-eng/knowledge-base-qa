## 🎯 推荐方案：云服务器部署（方案A）

### 第一步：准备服务器
1. **购买服务器**：腾讯云/阿里云轻量应用服务器
   - 配置：2核2G/3M带宽/40GB SSD
   - 系统：Ubuntu 22.04 LTS
   - 费用：约¥24-48/月

2. **获取访问信息**：
   ```
   服务器IP: 你的服务器公网IP
   用户名: ubuntu (默认)
   密码/密钥: 购买时设置
   ```

### 第二步：一键部署（5分钟完成）

#### 方法1：SSH远程部署
```bash
# 1. 连接到服务器
ssh ubuntu@你的服务器IP

# 2. 下载部署脚本
wget https://raw.githubusercontent.com/你的用户名/knowledge-base-qa/main/deploy.sh

# 3. 执行部署
sudo bash deploy.sh
```

#### 方法2：手动部署
```bash
# 1. 更新系统
sudo apt update && sudo apt upgrade -y

# 2. 安装Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs

# 3. 安装PM2
sudo npm install -g pm2

# 4. 克隆代码
git clone https://github.com/你的用户名/knowledge-base-qa.git
cd knowledge-base-qa

# 5. 安装依赖
npm run install:all

# 6. 构建前端
npm run build

# 7. 配置环境变量
cp .env.example .env
# 编辑.env文件，填入API密钥

# 8. 启动服务
pm2 start "npm run server" --name "knowledge-backend"
pm2 start "npm run client" --name "knowledge-frontend"
pm2 save
```

## 🌐 域名配置（可选）

### 1. 购买域名
- 推荐：阿里云/腾讯云域名注册
- 费用：约¥10-50/年（如：your-test.com）

### 2. 解析设置
```
记录类型: A
主机记录: @ 或 www
记录值: 你的服务器IP
TTL: 10分钟
```

### 3. Nginx配置
```bash
# 安装Nginx
sudo apt install -y nginx

# 配置站点
sudo cp nginx-knowledge-base.conf /etc/nginx/sites-available/knowledge-base
sudo ln -s /etc/nginx/sites-available/knowledge-base /etc/nginx/sites-enabled/

# 测试并重启
sudo nginx -t
sudo systemctl restart nginx
```

### 4. SSL证书（免费）
```bash
# 安装Certbot
sudo apt install -y certbot python3-certbot-nginx

# 获取证书
sudo certbot --nginx -d your-domain.com -d www.your-domain.com
```
