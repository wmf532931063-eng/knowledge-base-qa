## 使用指南

### 支持的问题格式
1. **基本出生日期**：我出生于1990年5月15日
2. **运势咨询**：我的生日是1988-08-20，运势如何？
3. **完整八字排盘**：帮我算一下2000年1月1日凌晨3点出生的八字
4. **命理分析**：生辰八字：1975年12月25日出生

### 支持的日期格式
- YYYY-MM-DD（1990-05-15）
- YYYY年MM月DD日（1990年5月15日）
- YYYY/MM/DD（1990/05/15）
- YYYY.MM.DD（1990.05.15）

### 支持的时辰格式
- X点（3点、15点）
- X时（3时、15时）
- 凌晨X点（凌晨3点）
- 下午X点（下午3点）

## 技术细节

### 核心函数
1. `convertSolarToLunar()` - 公历转农历和八字排盘
2. `extractBirthDate()` - 从问题提取日期信息
3. `processBirthDateQuestion()` - 处理出生日期问题
4. `containsBirthDate()` - 检测是否包含出生日期

### 处理流程
1. 用户提问 → 2. 检测出生日期 → 3. 提取日期和时辰 → 4. 农历转换 → 5. 八字排盘 → 6. 问题增强 → 7. 知识库检索 → 8. AI回答
